# Phase 3 결정: 대안 모델 평가

## 조사 완료

### MT3 (Google Magenta)
- **장점:** SOTA, Transformer 기반
- **단점:** JAX/Flax (PyTorch 아님), 통합 복잡도 높음
- **PyTorch 포트:** 비공식 (40 stars), 신뢰도 낮음
- **YourMT3:** 유망하지만 framework 불명
- **예상 성공률:** 60%
- **예상 시간:** 3-4일

### Onsets and Frames
- **장점:** PyTorch, Piano 특화, 50.22 F1 (SOTA 2018)
- **단점:** Pre-trained 모델 없음, Training 2주 소요
- **예상 성공률:** 50%
- **예상 시간:** 2주

## 현실적 평가

### 시간 투자 대비 효과
1. **MT3 시도:** 3-4일 투자 → 60% 성공 → 성능 불명
2. **Onsets and Frames:** 2주 투자 → 50% 성공 → 50.22 F1 예상
3. **현재 상태:** 36.66% (Phase 2.1 완료)

### 85% 목표 달성 가능성
- **MT3:** 불명 (benchmark 없음)
- **Onsets and Frames:** 50.22 F1 ≠ 85% similarity
- **현실:** 두 모델 모두 85% 보장 못함
