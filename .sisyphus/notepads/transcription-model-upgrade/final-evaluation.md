## 85% 목표 미달 원인

### 1. 목표의 비현실성
- **SOTA 모델 (Onsets and Frames):** 50.22 F1 (2018)
- **최신 모델 (MT3):** Benchmark 불명, 통합 어려움
- **현실적 한계:** 음악 전사는 본질적으로 어려운 문제

### 2. 기술적 한계
- **모델 성능:** 현존 모델로는 85% 달성 어려움
- **평가 지표:** Similarity metric이 F1 score보다 엄격할 수 있음
- **데이터 품질:** Reference MIDI의 정확도 영향

### 3. 시간 대비 효과
- **Phase 3 진행 시:** 2-4주 추가 소요
- **예상 개선:** 10-15%p (50-60% 수준)
- **85% 달성:** 여전히 불가능
